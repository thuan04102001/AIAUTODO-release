# AI AUTODO Release

Public release-only repository for AI AUTODO update manifests and release assets.

This repository intentionally does not contain application source code.

## Public files

- `update/stable.json`: stable app update manifest.
- `update/gpu-pack.json`: optional GPU Pack manifest.
- GitHub Releases: installer, quick patch, and GPU Pack assets.

## Current stable assets

Release `v1.0.23` should contain:

- `AI.AUTODO.Setup.1.0.23.exe`
- `AI.AUTODO.App.Patch.1.0.23.zip`

Release `v1.0.20` should contain GPU Pack parts:

- `AI.AUTODO.GPU.Pack.1.0.20.zip.part1`
- `AI.AUTODO.GPU.Pack.1.0.20.zip.part2`

Do not commit source code or private configuration to this repository.
